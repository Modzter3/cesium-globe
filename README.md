# CesiumJS Globe вЂ” Next.js

Interactive 3D globe with Cesium Ion imagery, world terrain, and a minimal dark HUD вЂ” built with Next.js 15 (App Router) and TypeScript.

---

## Features

- **3D world terrain** via Cesium Ion World Terrain
- **Imagery switcher** вЂ” Satellite, OpenStreetMap Streets, Night Earth
- **Mouse coordinate readout** вЂ” lat/lon + camera altitude
- **Fly home** button to reset the camera
- Cesium static assets auto-copied to `public/` via webpack
- Fully SSR-safe вЂ” the Cesium Viewer is dynamically imported client-side only

## Quick Start

```bash
# 1. Install dependencies (first time only)
npm install

# 2. Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

> **Note:** On the first `npm run dev`, webpack copies ~180 MB of Cesium static
> assets into `public/cesium/`. This takes 20вЂ“40 seconds. Subsequent starts are
> instant because the files are cached.

## Cesium Ion Token

The project ships with a demo token. For production use, replace the token in
`components/GlobeViewer.tsx`:

```ts
Cesium.Ion.defaultAccessToken = "YOUR_TOKEN_HERE";
```

Get a free token at [ion.cesium.com](https://ion.cesium.com) в†’ Account в†’ Access Tokens.

## Customisation

### Change the starting view

In `components/GlobeViewer.tsx`, find the `flyTo` call in `useEffect` and edit
the longitude, latitude, and height (in metres):

```ts
viewer.camera.flyTo({
  destination: Cesium.Cartesian3.fromDegrees(-98.5, 39.5, 8_000_000),
  duration: 0,
});
```

### Add a marker / entity

After the viewer is created, push an entity:

```ts
viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(-95.5, 29.7), // Houston
  point: { pixelSize: 10, color: Cesium.Color.RED },
  label: { text: "Houston, TX", font: "14px Inter" },
});
```

### Enable 3D buildings

```ts
const tileset = await Cesium.createOsmBuildingsAsync();
viewer.scene.primitives.add(tileset);
```

## Adding Custom Terrain

Replace `Cesium.Terrain.fromWorldTerrain()` with any
`CesiumTerrainProvider` or leave it as `false` for a flat ellipsoid.

## Troubleshooting

| Symptom | Fix |
|---|---|
| Black screen on first load | Wait for webpack to finish copying Cesium assets (~30 s) |
| `window is not defined` | Make sure the component is imported with `ssr: false` вЂ” already done in `page.tsx` |
| Slow terrain / imagery | Cesium Ion free tier is rate-limited; add your own token |
| TypeScript errors on `Cesium.*` | Run `npm install` again to ensure `@types` are installed |
